return문

반환하는 값의 명확한 의미를 알기 위해 괄호를 사용하지 않습는다.

표 6-19 return문 작성 예
잘못된 예 올바른 예
return (age <= 7 ? "baby" : "teenager");
var generation = (age <= 7 ? "baby" : "teenager");
return generation;

if 문

한 문장이라도 중괄호를 포함합니다.

표 6-20 if문 작성 예
잘못된 예 올바른 예
if (person != null)
    work();
if (person != null) {
    work(); }

for 문

변수는 for문의 초기화 구문 이전에 선언합니다.

표 6-21 for문 작성 예
잘못된 예 올바른 예
for (var i = 0; i < 10; i++) {

}
var I = 0,
        len = 10;
for (i = 0; i < len; i++) {

}
주의
  • 보통 객체 순회에는 for-in을 사용합니다.
  • 자바스크립트에서는 배열(Array)도 객체(Object)이지만 배열을 순회할 때 for-in을 사용해서는 안됩니다.
  • for-in은 프로토타입 체인에 있는 모든 프로퍼티를 순회하기 때문에 for를 사용할 때보다 보통 10배 이상 느리고, index 순서대로 배열을 순회한다고 보장할 수 없다.
  • 순회 순서는 브라우저에 따라 다를 수 있습니다.
표 6-22 배열을 순회할 때의 for문 사용 예
잘못된 예 올바른 예
var scores = [70, 75, 80, 61, 89, 56, 77, 83, 90, 93, 66];
var total = 0;
var score;

for (score in scores) {
    total += scores[score];
}
var scores = [70, 75, 80, 61, 89, 56, 77, 83, 90, 93, 66];
var total = 0; var i = 0;
var len = scores.length;

for (; i < len; i += 1) {
    total += scores[i];
}

switch문

  • case는 switch를 기준으로 한 단계 들여쓰기 합니다.
  • 첫 번째 case를 제외한 case와 default 사이에는 한 줄의 공백이 있어야 합니다.
  • default를 제외한 case는 return, break, throw 중 하나로 마쳐야 하고 다음 case로 의도적으로 넘기는 부분은 주석처리를 합니다.
  • default를 의도적으로 생략했으면 주석처리 합니다.
표 6-22 switch문 작성 예
잘못된 예 올바른 예
switch (num) {
    case 0:
        break;
    case 1:
        break;
    default:
        break;
}
switch (num) {
    case 0:
    // 다음 case와 같은 동작 처리

    case 1:
        break;
    default:
        break;
}
주의

switch는 if에 비해 간결한 코드를 작성할 수 있지만, 비교적 많은 메모리를 사용하고 저버전 IE에서는 처리 성능도 좋지 않다. 최신 버전 IE와 크롬과 같은 고성능 브라우저만 지원하는 서비스라면 이런 부분을 크게 고민하지 않아도 되겠지만, 저버전 IE를 포함한 다양한 환경을 지원해야 하는 서비스라면 switch문의 득과 실에 대해 따져볼 필요가 있습니다.

  • if문은 원하는 조건이 나올 때까지 순차적으로 모든 비교문을 순회하면서 비교
  • switch문은 jump-table을 사용하여 한 번에 원하는 곳으로 이동
  • if문은 조건 문의 개수만큼 O(n)의 시간 복잡도를 갖게 되어 성능의 단점
  • switch문은 case의 개수만큼 jump-table을 차지하므로 메모리의 단점
    • 저버전 IE에서는 성능에도 문제가 있음
    • case의 조건이 정수형이 아닌 경우(문자열 또는 연산)라면 jump-table을 쓸 수 없어 성능에도 이득 없음
표 6-23 조건문 개수에 대한 if, switch문 작성 예
잘못된 예 올바른 예
switch(foo) {
    case 'alpha':
        alpha();
        break;     case 'beta':
        beta();
        break;

    default:         ...
        break;
}
// 객체 리터럴 방식
var switchObj = {
    alpha: function() {
        ...
    },
    beta: function() {
        ...
    },
    _default: function() {
        ...
    } };
switchObj[foo](args);
// 모듈 방식
var switchModule = (function() {
    return {
        alpha: function() {
            ...
        },
        beta: function() {
            ...
        },
        _default: function() {
            ...
        }
    };
})();
switchModule[foo](args);
