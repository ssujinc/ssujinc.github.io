참고사항

  • 이 문서는 마크업 언어를 다루는 개발자를 대상으로 합니다.

기본규칙

미디어 타입 선언을 위한 명령문(중괄호 포함)은 다음과 같이 세부 스타일을 지정하는 명령문과 줄로 구분합니다. 인쇄를 위한 미디어 타입은 기본 CSS 파일의 가장 아랫부분에 선언하며 별도의 CSS 생성은 허용하지 않습는다.

…
… /* For Print */
@media print{
#header,.snb,.aside{display:none}
}
참고

미디어 타입은 각종 미디어(프린터, 모바일, 보조공학기기 등)에 대응하는 별도의 CSS 코드를 작성할 수 있게 합니다. CSS 2.1 Specification 에 따라 대응 가능한 미디어 타입은 아래와 같으며, 가장 범용적으로 사용하는 타입은 print 타입이다.

  • all (모든 출력 장치)
  • aural (음성 출력)
  • braille (점자 출력)
  • handheld (포켓, 모바일)
  • print (인쇄)
  • projection (투사 장치)
