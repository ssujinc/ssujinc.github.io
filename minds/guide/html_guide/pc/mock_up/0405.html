주석

A. 기본 형식
HTML 주석의 시작과 종료는 아래와 같이 표기하며, 기본 형식에 맞게 작성합니다.

시작 주석 <!!-- #주석내용 -->
종료 주석 <!!-- //#주석내용 -->

  • 주석 기호와 주석 내용 사이에는 반드시 공백 두 칸이 있어야 합니다.
  • 주석 기호와 주석 내용 사이의 줄 바꿈은 허용하지 않습는다. 단, 주석 내용 긴 줄 바꿈은 허용합니다.
  • 주석 내용 앞에는 id와 class 구분을 표시합니다.
표 3-8 주석 기본 형식 예
잘못된 예 올바른 예
<!!--#GNB-->
 <!!-- #GNB -->

<!!-- //#GNB -->

주의
  • 마크업과 개발의 편의를 위해 작성한 주석은 실제 서비스를 적용할 때 반드시 삭제합니다.
B. 레이아웃 및 콘텐츠 영역의 주석 표기
wrap을 제외한 레이아웃과 독립된 콘텐츠 영역의 끝에 주석을 표기하며, 레이아웃은 id 이름과 동일하게 주석을 넣는다. 독립된 콘텐츠 영 역의 주석 표기는 선택 사항이다.
HTML 코드와 주석은 줄로 분리해야 합니다.

<!-- #content -->
<div id="content">
    <!-- .namecard -->
    <div class="namecard"> ... </div>
    <!-- //.namecard -->
</div>
<!-- //#content -->

C. 개발 적용과 관련된 주석 표기
개발 적용과 관련된 주석은 해당되는 영역 위에 표기하며, 종료 주석은 표기하지 않습는다. 주석 앞에는 [D]라는 말머리를 사용하여 담당 개 발자가 반드시 확인할 수 있도록 합니다. 주석이 두 줄 이상이 되더라도 주석 기호는 한 번만 표기합니다.
표 3-8 주석 기본 형식 예
잘못된 예 올바른 예
<!-- 케이스별 클래스 변화 -->
<!-- 의사 : my_doctor 변호사 : my_lawyer --> 		<!-- [D] 케이스별 클래스 변화
의사 : my_doctor 변호사 : my_lawyer -->
<img src="btn.gif" width="100" height="100" alt="버튼">
<!-- 숨김처리
<img src="btn_on.gif" width="100" height="100" alt=“버튼 ">
//숨김처리 --> 		<img src="btn.gif" width="100" height="100" alt=“버튼">
<!-- [D] 활성된 버튼은 파일명에 _on추가
<img src="btn_on.gif" width="100" height="100" alt=“버튼"> -->
