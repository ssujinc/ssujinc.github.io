기본 규칙

CSS를 새로 작성할 때는 아래 파일을 기본으로 합니다. 초기 파일에는 스타일 속성 초기화 문장이 포함되어 있으며, 초기화 문장은 변경할 수 없다. CSS 작성 시 반드시 회사명/팀명/작성자/날짜를 기입합니다.

@charset "utf-8";
/* MINDsLab. 서비스개발. YMJ. 20190830 */

/* [reset css] */
* {font-family:"Spoqa Han Sans", sans-serif;}
html,body {margin:0; padding:0; width:100%; height:100%; font-family:"Spoqa Han Sans", sans-serif; font-size:12px; color:#666; line-height:18px; background:none;}
div,dl,dt,dd,ul,ol,li,h1,h2,h3,h4,h5,h6,pre,form,fieldset,input,textarea,p,blockquote,th,td,legend,em,span {margin:0; padding:0; font-family:"Spoqa Han Sans", sans-serif;}
table {border-spacing:0; border-collapse:collapse; font-family:"Spoqa Han Sans", sans-serif;}
th, td {word-break:break-all;} ...

모바일에서 초기 파일은 브라우저에 따라 w.css, e.css 파일을 작성합니다.

w.css
@charset "utf-8";
/* MINDsLab. 서비스개발. YMJ. 20190830 */

/* [reset css] */
* {font-family:"Spoqa Han Sans", sans-serif;}
html,body {margin:0; padding:0; width:100%; height:100%; font-family:"Spoqa Han Sans", sans-serif; font-size:12px; color:#666; line-height:18px; background:none;}
div,dl,dt,dd,ul,ol,li,h1,h2,h3,h4,h5,h6,pre,form,fieldset,input,textarea,p,blockquote,th,td,legend,em,span {margin:0; padding:0; font-family:"Spoqa Han Sans", sans-serif;}
table {border-spacing:0; border-collapse:collapse; font-family:"Spoqa Han Sans", sans-serif;}
th, td {word-break:break-all;}
form,fieldset {display:block;}
form,fieldset,img {border:none;}
address,caption,cite,code,dfn,em,th,var {font-style:normal; font-weight:normal;}
ol,ul,li {list-style:none;}
abbr,acronym {border:none;}
hr {display:none;}
a {text-decoration:none; color:#333; cursor:pointer; font-family:"Spoqa Han Sans", sans-serif;}
a:link,a:visited {text-decoration:none;}
a:hover,a:active,a:focus {color:#000; text-decoration:none;}
textarea {overflow:auto; resize:none;}
html:first-child select {min-height:20px;} /* opera 9 & below fix */
option {padding-right:6px;} /* firefox fix */
legend {position:absolute; top:0; left:0; width:0; height:0; overflow:hidden; visibility:hidden; font-size:0; line-height:0;} /* for screen reader */
pre {word-wrap:break-word;}
label {font-family:"Spoqa Han Sans", sans-serif;}
h1,h2,h3,h4,h5,h6 {font-family:"Spoqa Han Sans", sans-serif;}
br {background:none !important;}
img {vertical-align:top;}
a:focus, input:focus, textarea:focus, button:focus {outline:none !important;}
button {cursor:pointer;}

/* input 기본 스타일 초기화 */
input {-webkit-appearance:none; -moz-appearance:none; appearance:none;}

/* input type number 에서 화살표 제거 */
input[type=number]::-webkit-inner-spin-button,
input[type=number]::-webkit-outer-spin-button {margin:0; -webkit-appearance:none; -moz-appearance:none; appearance:none;}
input[type=number] {-moz-appearance:textfield;}
/* Select box 스타일 초기화 */
select {-webkit-appearance:none; -moz-appearance:none; appearance:none;}
