참고사항

  • 이 문서는 마크업 개발자가 소스 코드 작성 시에 따라야 할 규칙을 기술합니다.
  • 이 문서는 마크업 언어를 다루는 개발자를 대상으로 합니다.

기본 규칙

A. 한 줄 주석
  • 하단의 코드를 설명하기 위해 독립된 줄에 주석 작성합니다.
  • 주석 앞의 코드를 설명하기 위해 줄 끝에 주석 작성합니다.
  • 여러 줄에 걸친 설명일 때는 사용하지 않습는다.
  • 독립된 줄에 한 줄 주석 작성 시에는 설명할 코드와 같은 단계 들여쓰기 사용하며, 이전 줄은 한 줄 비운다.
표 6-12 한줄 주석 사용 예
잘못된 예 올바른 예
if (age > 20) {
    // 이 줄은 나이가 20살 이상일 때만 실행됩니다.
    showAdultMovie();
}
if (age > 20) {

    // 이 줄은 나이가 20살 이상일 때만 실행됩니다.
    showAdultMovie();
}
if (age > 20) {

    // 이 코드는 만으로 나이가 20세 이상인 것이 아니고
    // 그냥 20세 이상 아니아니 이게 아니고
    // 아 아닙니다 만으로 나이가 20세 이상인
    // 사람을 판별하기 위한 조건문입니다
    showAdultMovie();
}
B. 여러 줄 주석
  • 첫 번째 줄은 여러 줄 주석을 여는 방법인 /*를 삽입하며, 다른 텍스트는 넣지 않습는다.
  • 그 다음 줄 부터는 *와 텍스트를 입력하는데 첫 째 줄의 *과 열을 맞춘다. *과 텍스트 사이에는 공백 하나를 넣어준다.
  • 마지막 줄은 여러 줄 주석을 닫는 방법인 */를 삽입하며 다른 텍스트는 넣지 않습는다.
표 6-13 여러 줄 주석 사용 예
잘못된 예 올바른 예

    *사람을 판별하기 위한 조건문입니다.
    */     showAdultMovie();
}
