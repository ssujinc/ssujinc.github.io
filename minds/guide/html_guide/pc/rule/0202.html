참고사항

  • 이 문서는 웹페이지 및 어플리케이션 제작 시 참고해야 할 사항에 대해 기술합니다.
  • 이 문서는 웹기술을 활용하여 서비스를 만드는 모든 이를 대상으로 합니다.
  • 해당 내용은 웹접근성 콘텐츠 제작 시 적용해야 할 필수 요건입니다. 그 밖 콘텐츠 제작 시 적용시키지 않아도 무관합니다.

공통영역 (헤더, 푸터)

퍼블리싱
  • 웹페이지 기본 언어 표시 (퍼블리싱→개발)
  • 본문 바로가기 링크 제공 (퍼블리싱→개발)
  • GNB, LNB, 푸터에 있는 모든 레이어 메뉴 기능을 키보드만으로 사용이 가능하도록 스크립트 수정 (퍼블리싱→개발)
  • 검색, 로그인 등의 입력폼에서 키보드 초점이 논리적으로 이동하도록 수정 (퍼블리싱→개발)
  • 더보기, 전체보기 링크를 코드상 해당 내용의 뒷부분으로 이동 (퍼블리싱→개발)
  • 모든 이미지에 alt 값 삽입 (퍼블리싱→개발)
  • 모든 입력폼과 label 텍스트 연결 (퍼블리싱→개발)
  • label 텍스트가 없는 입력폼에 title값 제공 (퍼블리싱→개발)
개발
  • 웹페이지 기본 언어 표시 (개발)
  • 각 페이지를 구분할 수 있는 제목 제공 (개발)
  • iframe의 용도를 설명하는 title 값 삽입 (개발)

페이지별 단순작업

퍼블리싱
  • 모든 이미지와 이미지맵에 alt 값 삽입 (퍼블리싱→개발)
  • 키보드 포커스를 안보이게 하는 CSS 수정 (퍼블리싱)
  • 스크롤이 필요한 콘텐츠를 키보드로 스크롤 할 수 있도록 수정 (퍼블리싱→개발)
  • 제목을 헤딩태그(h1~h6)로 마크업 (퍼블리싱→개발)
  • 탭으로 바뀌는 각각의 콘텐츠에 제목 제공 (퍼블리싱→개발)
  • 모든 데이터 테이블에 <caption> 및 summary 값 제공 (퍼블리싱→개발)
  • 모든 데이터 테이블을 <th>, colscope속성을 사용하여 이해하기 쉽게 구성 (퍼블리싱→개발)
  • 모든 입력폼과 label 텍스트 연결 (퍼블리싱→개발)
  • label 텍스트가 없는 입력폼에 title값 제공 (퍼블리싱→개발)
개발
  • 모든 텍스트 아이콘, 버튼 이미지에 alt 값 삽입 (개발)
  • 별평점 이미지에 alt 값(5점 만점에 3점) 삽입 (개발)
  • alt값과 title값이 동일한 경우 title 제거 (개발, 퍼블리싱)
  • iframe의 용도를 설명하는 title 값 삽입 (개발)
  • 키보드 포커스를 강제로 이동시키거나 사라지게 하는 스크립트 제거 (개발)
  • 새창으로 뜨는 링크에 '새창으로 열림' title 값 제공 (개발, 퍼블리싱)
  • 문법오류 수정 (개발, 퍼블리싱)
디자인
  • 본문 텍스트와 배경간의 명도대비 수정 (디자인→퍼블리싱)
기획
  • 시각, 청각에 무관하게 인식할 수 있도록 지시사항 수정 (기획→개발)
  • 부적절한 링크텍스트 수정 (기획→개발)

페이지별 고급작업

퍼블리싱
  • 복잡한 내용의 이미지에 대한 대체 컨텐츠 생성 (퍼블리싱→개발)
  • 키보드 초점이 논리적으로 이동하도록 수정 (퍼블리싱→개발)
  • 제목 옆에 있는 더보기 링크를 코드상 해당 내용의 뒷쪽에 제공 (퍼블리싱→개발)
개발
  • 상품, 배너 이미지 alt 값 삽입을 위한 관리도구 개발 (개발)
  • onfocus, onchange로 새창이 뜨거나 서식이 제출되는 스크립트 수정, 필요한 경우 버튼 추가 (개발)
디자인
  • 색에 무관하게 콘텐츠를 인식할 수 있도록 수정 (디자인→퍼블리싱→개발)
  • 자동 롤링되는 콘텐츠에 정지버튼, 탐색기능 제공 (디자인→퍼블리싱→개발)
기획
  • 모든 동영상 콘텐츠에 자막 또는 대본 제공 (기획→디자인→퍼블리싱→개발)
  • 시간 제한이 있는 콘텐츠의 응답시간을 조절할 수 있는 기능 제공 (기획→디자인→퍼블리싱→개발)
  • 서식 전송 기능을 셀렉트박스만으로 제공하는 경우 전송 버튼 제공 (기획→디자인→퍼블리싱→개발)
  • 페이지 링크 기능을 셀렉트박스만으로 제공하는 경우 레이어 방식으로 제공 (디자인→퍼블리싱→개발)
