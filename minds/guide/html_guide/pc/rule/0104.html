ID Selector 예약어
구분 예약어 설명

#contents 본문 영역

Class Selector

  • 체인 선택자의 개수는 3개까지 권장합니다.
  • 클래스 예약어는 상황에 맞게 선택적으로 참고합니다.
  • 레이아웃에 한하여 정의된 예시이며, 자세한 내용은 설계가이드를 참고합니다.

구분 예약어 설명

.snb 서브메뉴 (Sub Navigation Bar)
.unb 기타메뉴 (Utility Navigation Bar)
.qnb 빠른메뉴 (Quick Navigation Bar)
.fnb 바닥글메뉴 (Footer Navigation Bar)
콘텐츠 레이아웃 .content 본문 영역
.titArea 본문 헤더 영역
