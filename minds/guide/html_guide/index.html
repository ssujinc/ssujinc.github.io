마크업 기준

문서 및 버전
HTML5, CSS3
HTML은 Hyper Text Markup Language의 약자로써 웹페이지를 만들기 위한 언어로 웹브라우저 위에서 동작하는 마크업 언어이다.
마인즈랩에서 서비스 하고자 하는 기술에 대해, 사용자 웹 환경과 변화하는 웹 생태계를 고려하여 최적으로 개발 구현가능한 HTML5, CSS3 기반으로 작성한다.
문서 언어
ko
PC나 모바일을 통해서 웹에 접근 하는 경우 웹접근성을 향상 시키기 위해 필요한 속성이다.
lang="ko"를 기재했을 경우, 스크린리더나 점자정보단말기, 모바일의 VoiceOver, TalkBack 등으로 웹에 접근하는 경우, 해당 언어에 알맞게 번역(점자,소리)되어 올바른 해석으로 정보 전달 될 수 있게 된다.
인코딩
UTF-8
문자 인코딩은 문자나 기호들의 집합을 컴퓨터에 저장, 사용할 목적으로 부호화하는 방법을 말한다.
한글과 영어만 사용하는 페이지에 대해 한글을 2바이트로 사용하는 문자집합인 EUC-KR 인코딩이 적합하다.
하지만 다양한 언어로 작성되는 환경과 한글 언어팩이 설치되지 않은 사용자 환경을 고려하여, 비록 한글을 3바이트로 인식하여 용량은 다소 커지지만 다국어 표현과 HTML4, XHTML, HTML5에 대한 대응이 가능하다.
